Kepler Web (moderno, sin plugins)

Este stub reemplaza al loader Shockwave. Aquí debe montarse el cliente HTML5/WebAssembly que hable el protocolo de Kepler.

Estado

Cliente moderno todavía pendiente de implementar. Usa esta página para probar configuración y bootstrap.

Parámetros soportados (query string)

sso (ticket SSO) ws (URL WebSocket, ej: wss://tu-dominio/ws) variables (URL de external_variables.txt) texts (URL de external_texts.txt)

Ejemplo: ?sso=ABC-123&ws=wss://ejemplo/ws&variables=/gamedata/external_variables.txt&texts=/gamedata/external_texts.txt
